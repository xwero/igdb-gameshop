# IGDB Gameshop

An experiment with issue driven AI development.

## The goal

- Import functionality for the IGDB data.
- Shopping basket API endpoints for
  - showing a basket
  - adding an item to a basket
  - deleting an item from a basket
- An API endpoint for salespeople to get data to use in a CRM solution.

## The method

Making an AI agent that checks different module descriptions to access the work that has to be done and generate the code from that assessment.

The idea is to use the git staged files as a selector between the work that has been done and the work that still needs to be done.
